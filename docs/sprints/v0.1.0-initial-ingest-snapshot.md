  one file with embedded YAML frontmatter that the parser handles
  cleanly (no warnings logged).
- Embedding throughput on M4 Pro with `Qwen/Qwen3-Embedding-0.6B`
  averaged roughly **1486 chunks / 70 s** end-to-end (pipeline + I/O
  + Qdrant upsert), i.e. ~21 chunks/s wall-clock. Pure embedding is
  faster but parser + scroll lookups dominate at this scale.
