# Opening hook

This file mirrors the brand voice samples that will be ingested into
SDET Brain. The frontmatter carries the metadata the future search
tools will filter by - tags, status, language, slug - while the body
is plain Markdown the chunker needs to split into one or two chunks
depending on length.

## How we use it

A voice sample documents how the brand sounds in a specific situation:
opener, mid-thread, sign-off. The chunker should preserve this header
hierarchy in `heading_path` so a query like "show me brand openers"
can rank by section, not just by file.
